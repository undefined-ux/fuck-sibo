## Build
```bash
cargo build --release
```

## Usage
```bash
Usage: fuck-sibo-rs.exe [OPTIONS] [COMMAND]

Commands:
  search  Search School ID by name.
  help    Print this message or the help of the given subcommand(s)

Options:
  -c <config>
  -t <thread>                     [default: 1]
  -u <user-name>
  -p <password>
  -s <school-id>
  -n <number-of-completions>      [default: 1]
  -h, --help                      Print help
```
